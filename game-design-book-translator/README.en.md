# Game Design Book Translator

**Parent project:** [GameDesignOS by Paranoia](../README.en.md)

Use this skill to translate, polish, structure, and QA-check English game design, game development, Game Studies, system design, numeric design, UX, and design theory materials into professional Chinese for designers, developers, producers, and researchers.

## Scope

- Short excerpts: produce natural Chinese with light terminology notes when useful.
- Chapters and long-form essays: stabilize core terminology before translating body text, captions, tables, and key lines.
- Whole books, PDFs, or OCR drafts: create `README.md`, `source_manifest.md`, `00-术语表.md`, chapter Markdown files, and `assets/`.
- Existing Chinese drafts: unify terminology, remove translationese, backfill captions, and produce QA notes.
- Public sharing: check source, authorization, and quotation boundaries before shaping public-facing output.
