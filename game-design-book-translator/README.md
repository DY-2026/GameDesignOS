# Game Design Book Translator

**父项目：** [ParanoiaSkills](../README.zh-CN.md)

**语言：** 简体中文 | [English](./README.en.md)

这是 `ParanoiaSkills` 里的一个可安装 skill，用于把游戏设计、游戏研发、Game Studies、系统设计、UX/design theory 材料翻译、精修和审校成专业中文。

## 能力边界

- 短摘录：直接给自然中文译文，必要时补术语说明。
- 单章/长文：先统一核心术语，再翻正文、图注、表格和关键句。
- 整本书/PDF/OCR：建立 `README.md`、`source_manifest.md`、`00-术语表.md`、逐章 Markdown 和 `assets/`。
- 已有中文译稿：做术语统一、去英文腔、补图注和审校记录。
- 公开传播：先判断来源、授权和引用比例，不默认生成可公开发布的全文译本。
