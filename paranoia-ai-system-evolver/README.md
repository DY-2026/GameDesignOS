## What This Skill Does

`Paranoia AI System Evolver` helps agents upgrade prompts, workflows, memory, schemas, RAG, tool routing, eval sets, and feedback loops without turning one successful case into an uncontrolled permanent rule.

It is built around seven controls:

- **Model compression:** explain more real tasks with a shorter, more reusable operating model.
- **Causal mediators:** expose the intermediate variables that can be observed, intervened on, and verified.
- **VOI:** decide what information is worth acquiring.
- **OODA:** keep the agent oriented during real work.
- **Evals:** test result quality, process quality, and evolution risk.
- **Human Gate:** require confirmation before high-impact changes.
- **Rollback:** keep every promoted change reversible.
