## Why This Release Matters

`ParanoiaSkills` already has portable skill packages. This launch pack makes the repository easier for strangers to understand, try, and safely contribute to within 10 minutes.

The release does not add a large new skill, does not introduce author real-project cases, and does not publish client or private-overlay material.

## Try It In 10 Minutes

Start here:

- [Try It in 10 Minutes](../docs/try-it-in-10-minutes.md)
- Copy one folder: `game-experience-analyzer/`, `game-concept-architect/`, or `paranoia-ai-system-evolver/`
- Trigger one skill with `$game-experience-analyzer`, `$game-concept-architect`, or `$paranoia-ai-system-evolver`
- Run `python scripts/validate_repo.py` from the repository root

## Public Example Policy Reminder

Public repository material must be synthetic, public, or explicitly cleared.

Do not submit private projects, client work, unreleased roadmaps, internal strategy, private screenshots, private recordings, local workspace paths, private overlays, or real project cases. Users may use real projects in their own private environment, but those materials must not be committed to this public repository.

If source status is unclear, mark it `needs_review`.

## Validation Commands

Run from the repository root:

```text
python scripts/validate_repo.py
python scripts/validate_skill.py game-experience-analyzer
python scripts/validate_skill.py game-concept-architect
python scripts/validate_skill.py paranoia-ai-system-evolver
```
