# GameDesignOS v1 Workspace Template

这是 Project-Ready GameDesignOS 的目标项目结构。它和 v0.9 模板分开保留，避免旧 workspace 被强制迁移。

```text
00-inbox/
01-decisions/
02-assumptions/
03-evidence/
04-experiments/
05-design-assets/
06-workflows/
07-learning/
08-exports/
.gamedesignos/
```

核心规则：一个设计判断只有连接到 Decision Object、明确假设、证据边界、实验或豁免、gate result 和人类承诺决策后，才能成为项目真相。
